# 🛒 React Redux Shopping Cart

A modern e-commerce frontend application built with React and Redux Toolkit. Users can browse products, add/remove items from the cart, and navigate through a clean and responsive interface.

## 🚀 Features

* 🛍️ Product listing from external API
* ➕ Add to cart functionality
* ➖ Remove items from cart
* 🔄 Global state management with Redux Toolkit
* 🧭 Page navigation with React Router
* ⚡ Fast and responsive UI with Tailwind CSS

## 🛠️ Technologies Used

* React
* Redux Toolkit
* React Router DOM
* Tailwind CSS
* JavaScript (ES6+)
* Fetch API

## 📦 API

This project uses an external API (Fake Store API) to fetch product data dynamically.

## 📌 How It Works

Products are fetched from an external API and displayed on the home page. Users can add items to the cart, which is managed globally using Redux Toolkit. The cart page displays selected items and allows users to remove them.
